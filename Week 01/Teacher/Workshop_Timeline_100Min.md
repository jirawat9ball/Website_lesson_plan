## 🎯 รายละเอียดและขั้นตอนการดำเนินการเชิงลึก (Step-by-Step Teacher Guide)

### 1. ช่วงปูสภาพแวดล้อมระบบและสร้างไฟล์ (20 นาที | 10:40 - 11:00 น.)
*   **กิจกรรมหลัก:** 
    1. นำนักเรียนเข้าโปรแกรม **Visual Studio Code (VS Code)**
    2. สอนวิธีสร้างโฟลเดอร์เก็บงานเฉพาะตัวของนักเรียน และเปิดโฟลเดอร์นั้นใน VS Code (File -> Open Folder)
    3. สร้างไฟล์ใหม่ตั้งชื่อว่า `index.html`
    4. ใช้คีย์ลัดสร้างโครงร่างอัตโนมัติ: พิมพ์เครื่องหมายตกใจ `!` แล้วกดปุ่ม `Tab` (หรือปุ่ม `Enter`) บนแป้นพิมพ์เพื่อเปิดโครงสร้าง HTML Boilerplate
*   **จุดเน้นย้ำและข้อระวังของครู:**
    *   ⚠️ **ความเข้าใจผิดยอดนิยม:** เด็กอาจสร้างไฟล์เป็น `index.html.txt` เพราะคอมพิวเตอร์ซ่อนนามสกุลไฟล์ไว้ ให้ครูช่วยเช็คสระสะกดของเด็กทีละหน้าจอ
    *   ⚠️ **แสดงผลภาษาไทย:** ตรวจสอบให้ชัวร์ว่าในบรรทัด `<head>` มีแท็ก `<meta charset="UTF-8">` เพื่อป้องกันปัญหาสระภาษาไทยพังหรือเพี้ยนในเบราว์เซอร์

### 2. ลงมือทำส่วนที่ 1: แกนหลักจัดอันดับ 3 หนังเด่น (25 นาที | 11:00 - 11:25 น.)
*   **กิจกรรมหลัก:** ให้นักเรียนสร้างชิ้นงานแกนหลัก **[Core Task]** จัดอันดับภาพยนตร์หรือการ์ตูนสุดประทับใจตลอดกาล 3 เรื่องแรกของชีวิต
*   **แท็กที่ใช้เรียนรู้:** 
    - `<h1>` (หัวข้อใหญ่หน้าเว็บรีวิว)
    - `<h3>` (หัวข้อย่อยระบุอันดับหนัง เช่น 🥇 อันดับ 1: Spirited Away)
    - `<p>` (เขียนบรรยายความรู้สึกหรือเหตุผลสั้นๆ ที่ชื่นชอบหนังเรื่องนั้น)
    - `<hr />` (สร้างเส้นคั่นสายตาระหว่างหนังแต่ละเรื่อง)
*   **เทคนิคการสอน:** ครูอาจปล่อยให้นักเรียนทดลองพิมพ์ และลองเปิดรันไฟล์ใน Chrome (โดยการดับเบิ้ลคลิกไฟล์ `index.html` ในโฟลเดอร์เครื่องคอมพิวเตอร์) เพื่อดูผลลัพธ์การขยายขนาดของตัวอักษรแต่ละระดับ

### 3. ลงมือทำส่วนที่ 2: ติดปีกความท้าทายระดับสูง (30 นาที | 11:25 - 11:55 น.)
*   **กิจกรรมหลัก:** อัปเกรดหน้าเว็บให้ดูมีความเป็นโปรแกรมเมอร์มืออาชีพขึ้นด้วยการเชื่อมต่อข้อมูลภายนอก **[Extra Task]**
*   **แท็กที่ใช้เรียนรู้:**
    - `<img>` (ฝังรูปภาพโปสเตอร์หนัง): สอนนักเรียนวิธีค้นหาโปสเตอร์หนังใน Google -> คลิกขวาเลือก **"Copy Image Address" (คัดลอกที่อยู่รูปภาพ)** -> นำมาวางที่ `src` ของแท็ก และกำหนดขนาดเช่น `width="200"` ไม่ให้ภาพล้นจอ
    - `<a>` (สร้างจุดไฮเปอร์ลิงก์): ค้นลิงก์ตัวอย่างหนังบน YouTube แล้วฝังผ่าน `<a href="ลิงก์Youtube" target="_blank">ดูตัวอย่างหนัง</a>`
*   **จุดเน้นย้ำและข้อระวังของครู:**
    *   ⚠️ **รูปภาพพัง:** ลิงก์รูปภาพของบางเว็บไซต์อาจบล็อกการดึงไปใช้ภายนอก (Hotlinking) ทำให้รูปขึ้นไอคอนเสียบนหน้าเว็บ วิธีแก้ไขคือให้เด็กเลือกหารูปจากเว็บใหม่ หรือเปลี่ยนไปหารูปจาก Wikipedia/IMDb ซึ่งมีความเสถียรมากกว่า
    *   ⚠️ **แท็กเดี่ยว:** เน้นย้ำให้นักเรียนเห็นว่า แท็กรูปภาพ `<img>` และเส้นคั่น `<hr />` เป็นแท็กเดี่ยวที่ไม่จำเป็นต้องมีแท็กปิด `</img>`

### 4. ตกแต่งลูกเล่นพิเศษสลับ "โรงภาพยนตร์จอมืด" (10 นาที | 11:55 - 12:05 น.)
*   **กิจกรรมหลัก:** สอนทริค CSS สั้นๆ เพื่อปรับโทนสีหน้าเว็บให้สวยงามสไตล์ Netflix หรือจอยักษ์ภาพยนตร์โหมดมืด (Dark Theme)
*   **แนวทางนำสอน:** ให้ครูเขียนคีย์เวิร์ดต่อไปนี้ขึ้นหน้าจอ เพื่อให้เด็กนำไปเขียนแทรกทับแท็ก `<body>` ปกติในบรรทัดที่ 7-8:
    ```html
    <body style="background-color: #1a1a1a; color: #e5e5e5; max-width: 800px; margin: 0 auto; padding: 40px;">
    ```
*   **ผลลัพธ์ทางจิตวิทยา:** การปรับเพียงจุดเดียวนี้จะช่วยให้นักเรียนเห็นผลลัพธ์ว่าหน้าเว็บที่เคยดูเรียบๆ น่าเบื่อ เปลี่ยนสไตล์ไปเป็นพรีเมียมได้ในพริบตา ปลูกฝังแรงจูงใจในการเขียนโค้ดต่อในบทเรียน CSS สัปดาห์ต่อๆ ไปได้อย่างยอดเยี่ยมครับ

### 5. กิจกรรมจัดแสดงและทดสอบผลงาน Mini Cinema Showcase (15 นาที | 12:05 - 12:20 น.)
*   **กิจกรรมหลัก:** จัดการแชร์ชิ้นงานและเรียนรู้เชิงสร้างสรรค์
*   **แนวทางการจัดกิจกรรม:**
    1. สั่งให้นักเรียนบันทึกไฟล์ (Save) แล้วหันหน้าจอคอมพิวเตอร์แชร์ให้เพื่อนข้างๆ ดูชิ้นงาน
    2. สลับกันคลิกลิงก์ชมวิดีโอตัวอย่าง ยืนยันความถูกต้องของลิงก์และภาพประกอบ
    3. ครูสรุปประเด็นแท็ก HTML พื้นฐานทั้งหมด และเชื่อมโยงเนื้อหาสู่การปูพื้นฐานการตกแต่งในสัปดาห์ถัดไป

## 🛠️ ข้อสังเกตและคู่มือแก้ปัญหาเฉพาะหน้าสำหรับคุณครู (Teacher Troubleshooting)

1.  **พิมพ์ไทยแล้วเป็นภาษาต่างดาว:**
    *   *สาเหตุ:* ลืมแท็ก `<meta charset="UTF-8">` หรือไฟล์ถูกบันทึกด้วย Encode อื่น
    *   *วิธีแก้:* ตรวจเช็คบรรทัดที่ 4-5 ในแท็ก `<head>` และกดเปลี่ยน encoding ที่มุมล่างขวาของโปรแกรม VS Code ให้เป็น UTF-8
2.  **รูปโปสเตอร์ขยายใหญ่เต็มหน้าจอ:**
    *   *สาเหตุ:* ลืมใส่มิติความกว้างให้รูปภาพ
    *   *วิธีแก้:* สั่งให้เขียน Attribute `width="200"` หรือ `height="300"` กำกับเข้าไปในแท็ก `<img>`
3.  **กดลิงก์แล้วหน้าเว็บแนะนำหนังตัวเดิมหายไป:**
    *   *สาเหตุ:* ไม่ได้ระบุเปิดแท็บใหม่ทำให้ทับเนื้อหาเดิม
    *   *วิธีแก้:* ใส่ข้อกำหนด `target="_blank"` กำกับเข้าไปในแท็ก `<a>` เพื่อบังคับเปิดแท็บอินเทอร์เน็ตใหม่เสมอ

## 🔑 สรุปแท็ก HTML ที่ประยุกต์ใช้ในคาบเรียนนี้ (HTML Tags Cheat Sheet)

*   `<!DOCTYPE html>`: ประกาศ HTML5
*   `<html>`: แท็กครอบทั้งหน้าเว็บ
*   `<head>`: แท็กส่วนหัวระบบ
*   `<meta>`: แท็กจัดการภาษาไทย (`charset="UTF-8"`)
*   `<title>`: ชื่อหัวแท็บเบราว์เซอร์
*   `<body>`: เนื้อหาหน้าเว็บที่แสดงผลจริง
*   `<h1>`-`<h6>`: แท็กหัวข้อใหญ่-ย่อย
*   `<p>`: แท็กย่อหน้าข้อความ
*   `<hr />`: แท็กสร้างเส้นคั่น
*   `<strong>`: แท็กเน้นอักษรหนา
*   `<div>`: แท็กบล็อกจัดกลุ่ม
*   `<img>`: แท็กฝังรูปภาพ (`src`, `width`)
*   `<a>`: แท็กสร้างลิงก์ (`href`, `target`)
*   `<footer>`: แท็กระบุส่วนท้ายเครดิต
