# สัปดาห์ที่ 1: Introduction to HTML

## กิจกรรมปฏิบัติ Workshop (เวลา 10:40 - 12:20 น.)
ยินดีต้อนรับสู่โลกแห่งการเขียนเว็บครับ! ในสัปดาห์นี้เราจะเริ่มสร้างหน้าเว็บไซต์หน้าแรกในชีวิต โดยใช้ภาษา HTML ในการกำหนดโครงร่างเนื้อหา

---

### 🎯 งานหลักที่ต้องส่ง (Core Task)
ให้นักเรียนจัดทำหน้าเว็บ **"Movie Ranking (แนะนำ 3 หนังโปรดตลอดกาล)"** ของตัวเอง
1. สร้างโครงร่างเว็บพื้นฐาน มีแท็ก `<html>`, `<head>`, `<title>`, `<body>` ครบถ้วน
2. ออกแบบหัวข้อหลักด้วยแท็ก `<h1>` และหัวข้อย่อยอันดับหนังด้วยแท็ก `<h3>`
3. เขียนบรรยายเหตุผลความประทับใจของตัวนักเรียนลงในแท็ก `<p>`
4. ใส่เส้นคั่นแบ่งสัดส่วนเนื้อหาให้สบายตาด้วยแท็ก `<hr />`

---

### 🚀 ความท้าทายพิเศษ (Extra Challenges)
หากนักเรียนทำส่วนหลักเสร็จรวดเร็ว สามารถอัปเกรดชิ้นงานเพิ่มเติมเพื่อให้ได้คะแนนพิเศษดังนี้:
1. **[Image Poster Challenge]**: ไปค้นหารูปภาพโปสเตอร์หนังจาก Google คัดลอกลิงก์มาวางในแท็ก `<img src="..." width="200">` เพื่อประดับหัวการ์ดรีวิวหนัง
2. **[Trailer Link Challenge]**: ค้นหาลิงก์วิดีโอตัวอย่างหนังใน Youtube แล้วนำมาฝังลงในแท็กสร้างลิงก์ `<a href="ลิงก์Youtube" target="_blank">ดูตัวอย่างหนังคลิกตรงนี้</a>`
3. **[Cinema Dark Theme]**: ตกแต่งพื้นหลังเว็บบล็อกด้วย CSS ด่วนเพื่อความเท่มีระดับ โดยแก้ไขแท็ก `<body>` ในบรรทัดที่ 7 ให้เป็น:
   `<body style="background-color: #1a1a1a; color: #e5e5e5; max-width: 800px; margin: 0 auto; padding: 40px;">`
