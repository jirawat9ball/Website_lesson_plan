### 1. 🎨 ส่วนที่ 1: จิตวิทยาของสีและรหัสสีดิจิทัล (15 นาที)
*   **แนวทางการอธิบายเรื่องการลงสี**:
    *   หน้าจอคอมพิวเตอร์สร้างสีจาก **"แสงของแม่สีทั้ง 3 สี" (RGB: Red, Green, Blue)** รวมกันเป็นแสงสีขาว
    *   **เปรียบเทียบรหัสสี 3 สไตล์**:
        1.  **HEX (#RRGGBB)**: รหัสเลขฐานสิบหก (0-9 และ A-F) เปรียบเสมือน **"พิกัดลับทางทหารของสี"** เช่น `#FF0000` คือ แดงล้วนไร้สีอื่นปน
        2.  **RGB (Red, Green, Blue)**: เปรียบเหมือน **"จานผสมแม่สีของช่าง"** ระบุปริมาณเนื้อสีตั้งแต่ 0 ถึง 255 (เช่น `rgb(255, 0, 0)`)
        3.  **HSL (Hue, Saturation, Lightness)**: เปรียบเหมือน **"แป้นหมุนปรับความรู้สึกจิตรกร"** (Hue = องศาวงล้อสี 0-360, Saturation = เปอร์เซ็นต์ความสดของสี, Lightness = เปอร์เซ็นต์ความมืดความสว่าง)
            *   *ข้อดี*: เป็นรหัสสีที่มนุษย์อ่านและปรับเปลี่ยนจินตนาการได้ง่ายที่สุดเมื่อเทียบกับรหัสอื่น!

### 2. ✍️ ส่วนที่ 2: ฟอนต์สร้างอารมณ์และแหล่งดึงฟอนต์ Google Fonts (15 นาที)
*   **แนวทางการเปรียบเทียบจิตวิทยาฟอนต์ (Typography)**:
    *   ฟอนต์เปรียบเสมือน **"เสื้อผ้าและน้ำเสียงของผู้พูด"**
        *   **Serif (ฟอนต์แบบมีหัว / มีฐานขีด)**: เปรียบเหมือน **"ผู้ใหญ่ในชุดสูทภูมิฐาน"** ให้ความรู้สึกดั้งเดิม น่าเชื่อถือ มีอำนาจ (เช่น Times New Roman, อักษรไทยแบบประดิษฐ์)
        *   **Sans-Serif (ฟอนต์ไม่มีหัว / ไม่มีขีด)**: เปรียบเหมือน **"วัยรุ่นในเสื้อยืดและกางเกงยีนส์"** ทันสมัย สะอาดตา อ่านง่ายบนหน้าจอคอมพิวเตอร์และมือถือ
    *   **Google Fonts**: บริการตู้คอนเทนเนอร์เก็บฟอนต์สาธารณะบนระบบคลาวด์ สอนวิธีการกดเลือกฟอนต์ คัดลอกแท็ก `<link>` แปะลงใน `<head>` และอ้างอิงผ่าน `font-family` ใน CSS

### 🔍 ส่วนที่ 3: เปิดกล่องเครื่องมือลับ Inspect Element (10 นาที)
*   **แนวทางการสอน**:
    *   ครูสอนกดปุ่ม **F12** บนคีย์บอร์ด หรือ **คลิกขวา -> ตรวจสอบ (Inspect)** บนเบราว์เซอร์
    *   แนะนำกล่องเครื่องมือของโปรแกรมเมอร์ (Developer Tools) เพื่อแอบดูโค้ดของเว็บไซต์ชื่อดังระดับโลก และทดลองแก้เปลี่ยนสีฟอนต์เล่นบนหน้าจอทันที (เน้นย้ำความขี้เล่น: "นี่เป็นการแก้ไขชั่วคราวในเครื่องเราเพื่อทดลองเฉยๆ พอกดรีเฟรชหน้าจอก็หายหมดนะ ไม่ใช่การแฮกเกอร์ระบบหลังบ้านของเขาจริงๆ!")

### 4. 🧠 ส่วนที่ 4: กิจกรรมทดสอบความเข้าใจด่วน (Quick Quiz) (5 นาที)
เช็กความพร้อมด้วย 3 คำถามด่วน:
1.  **คำถาม 1**: หากต้องการสีที่ดูพาสเทล (สว่างสดใสแต่อ่อนนุ่ม) การใช้รหัสสีใดระหว่าง HEX, RGB, HSL จะอำนวยความสะดวกในการปรับแต่งให้สว่างขึ้นได้ง่ายที่สุด? *(แนวตอบ: HSL เพราะแค่หมุนตัวเลข Lightness ไปแถวๆ 80-90% ก็จะได้สีโทนสว่างพาสเทลทันที)*
2.  **คำถาม 2**: เว็บไซต์ข่าวสารที่เป็นทางการและต้องการความน่าเชื่อถือสูงมากๆ ควรพิจารณาเลือกใช้ฟอนต์ตระกูลใดเป็นหลัก? *(แนวตอบ: Serif)*
3.  **คำถาม 3**: เครื่องมือ "Inspect Element" (คลิกขวา -> ตรวจสอบ) มีประโยชน์ต่อการเรียนรู้การเขียนเว็บอย่างไรบ้าง? *(แนวตอบ: ช่วยให้จำลองทดลองเปลี่ยนสี ขนาดตัวอักษร หรือสไตล์ของโค้ดดูได้ทันทีโดยไม่ต้องไปแก้ไฟล์จริง แล้วค่อยนำค่าที่ชอบกลับไปบันทึกลงในไฟล์โค้ด)*
