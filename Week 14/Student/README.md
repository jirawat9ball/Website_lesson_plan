# สัปดาห์ที่ 14: Final Project (Start) & Web Hosting

## กิจกรรมปฏิบัติ
ออกแบบโครงสร้างหน้าแรกของพอร์ตโฟลิโอ และอัปโหลดโปรเจกต์เดิมขึ้นเว็บไซต์จริงบนอินเทอร์เน็ต

### งานที่ต้องทำ (Core)
1. **ร่าง Wireframe** ลงกระดาษเพื่อกำหนดพื้นที่ Navbar, รูปโปรไฟล์แนะนำตัว, รายละเอียดทักษะ, คาร์ดแสดงผลงาน และช่องทางการติดต่อ
2. **อัปโหลดโปรเจกต์สัปดาห์ที่ 3 (Personal Wiki)** ขึ้นระบบอินเทอร์เน็ตจริงโดยใช้บริการฟรีของ **GitHub Pages**
   - ศึกษาขั้นตอนการอัปโหลดได้จากคู่มือ `Guide_GitHub_Pages.md` ในโฟลเดอร์ของสัปดาห์นี้

### งานเพิ่มเติม (Extra Challenge)
*   **สลับกันล่าลิงก์พัง (Broken Link Hunting)**: แลกเปลี่ยนลิงก์เว็บจริงกับเพื่อน เพื่อทดลองคลิกเมนูต่างๆ และช่วยเช็คคำสะกดของ Path ลิงก์ที่อาจจะเสียหาย (Error 404)
