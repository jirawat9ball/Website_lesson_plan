# Project Guide: Web Portfolio (Online Portfolio)

คอนเซปต์: "สร้างตัวตนบนโลกออนไลน์ ให้พร้อมยื่นมหา'ลัยด้วย Code ของเราเอง"

## 1. โครงสร้างของเว็บไซต์ (Site Map)
เว็บไซต์ต้องมีอย่างน้อย 3-4 ส่วนหลัก (จะแยกเป็นหน้าหรือทำเป็น Single Page ที่มีเมนูกดเลื่อนก็ได้):
- **หน้าแรก (Home / Hero Section)**: ภาพลักษณ์ที่น่าประทับใจ, สโลแกน, ปุ่ม Call to Action
- **เกี่ยวกับฉัน (About Me / Resume)**: ประวัติการศึกษา, ทักษะที่ถนัด (ใช้ Bootstrap Progress Bars หรือ Icons)
- **คลังผลงาน (Portfolio / Projects)**: ใช้ Bootstrap Cards จัดเรียงผลงานในรูปแบบ Grid
- **ช่องทางการติดต่อ (Contact)**: ลิงก์ Social Media, อีเมล, เบอร์โทรศัพท์

## 2. สิ่งที่ต้องมีในโปรเจค (Technical Checklist)
- [ ] Custom Font: ใช้ Google Fonts ที่สะท้อนบุคลิกตัวเอง
- [ ] Bootstrap Navbar: เมนูที่คลิกแล้วเลื่อนไปยังส่วนต่างๆ ของหน้าเว็บ
- [ ] Responsive Grid: คอมพิวเตอร์ 3 คอลัมน์, มือถือ 1 คอลัมน์
- [ ] Hover Effects: เมื่อเอาเมาส์ชี้ที่รูปผลงานหรือปุ่ม ต้องมีการเปลี่ยนแปลง
- [ ] Online Link: เว็บต้องออนไลน์ผ่าน GitHub Pages
