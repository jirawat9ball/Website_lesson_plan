## 🎯 โจทย์ภารกิจ: [Project] Personal Wiki (5 หน้า)

ให้นักเรียนสร้างเว็บไซต์ประวัติและผลงานส่วนตัวขนาด 5 หน้าเพจ โดยจัดกลุ่มไฟล์ออกเป็น 2 ชั้นโฟลเดอร์ตามโครงสร้างด้านล่างนี้:

### โครงสร้างโฟลเดอร์เว็บไซต์:
```
Student/
├── index.html (หน้าแรก)
├── about.html (เกี่ยวกับฉัน)
├── contact.html (ติดต่อฉัน)
└── hobbies/
    ├── hobbies.html (งานอดิเรกหลัก)
    └── gallery.html (คลังภาพงานอดิเรก)
```

---

### 🔗 ภารกิจหลักที่ 1: การเขียน Relative Path เชื่อมข้ามหน้า (Core Mission)

1.  **ในหน้าเว็บที่อยู่ด้านนอก (index.html, about.html, contact.html)** ให้เขียนลิงก์นำทางในแท็ก `<nav>` ดังนี้:
    ```html
    <nav>
        <a href="index.html">หน้าแรก</a> | 
        <a href="about.html">เกี่ยวกับฉัน</a> | 
        <a href="contact.html">ติดต่อฉัน</a> | 
        <a href="hobbies/hobbies.html">งานอดิเรก</a>
    </nav>
    ```
2.  **ในหน้าเว็บที่ซ้อนอยู่ในโฟลเดอร์ย่อย (hobbies/hobbies.html, hobbies/gallery.html)** ลิงก์นำทางจะต้องเขียนเพื่อถอยกลับออกไปข้างนอกด้วยการใช้สัญลักษณ์ `../` นำหน้าชื่อไฟล์รูท:
    ```html
    <nav>
        <a href="../index.html">หน้าแรก</a> | 
        <a href="../about.html">เกี่ยวกับฉัน</a> | 
        <a href="../contact.html">ติดต่อฉัน</a> | 
        <a href="hobbies.html">งานอดิเรก</a> | 
        <a href="gallery.html">คลังภาพงานอดิเรก</a>
    </nav>
    ```

---

### 🔗 ภารกิจหลักที่ 2: งานส่วนขยายพิเศษ (Extra Challenge)
1.  **แทรกวิดีโอแนะนำตัว** ในหน้าเกี่ยวกับฉัน (`about.html`) โดยใช้แท็ก `<video src="https://www.w3schools.com/html/mov_bbb.mp4" controls width="400"></video>`
2.  **แทรกเพลงประกอบ** ในหน้างานอดิเรก (`hobbies/hobbies.html`) โดยใช้แท็ก `<audio src="https://www.w3schools.com/html/horse.mp3" controls></audio>`
3.  **แทรกภาพถ่ายกิจกรรม** ในหน้าคลังภาพ (`hobbies/gallery.html`) ผ่านแท็ก `<img>` (เช่น `https://picsum.photos/300/200?random=1`)
4.  *ข้อแนะนำ*: อย่าลืมพิมพ์ Attribute `controls` ในแท็กมีเดียเด็ดขาดเพื่อไม่ให้เครื่องเล่นไม่มีปุ่มควบคุม!

## 🔍 รายการเช็คความถูกต้องก่อนส่ง (Self-Check Checklist)
- [ ] ชื่อไฟล์ทั้งหมดสะกดตัวอักษรพิมพ์เล็กทั้งหมด ไม่มีอักขระพิเศษหรือพิมพ์ใหญ่ปะปน
- [ ] โฟลเดอร์ย่อยตั้งชื่อว่า `hobbies/` อยู่ตำแหน่งถูกต้องเคียงข้างไฟล์หลัก
- [ ] เมื่อกดคลิกลิงก์เมนูนำทางในทุกหน้า (ทั้ง 5 หน้า) สามารถเปลี่ยนหน้าไปยังหน้าอื่นได้ครบถ้วนโดยไม่ขึ้นหน้า Error 404
- [ ] วิดีโอและไฟล์เสียงเล่นได้ปกติ และแสดงแถบปุ่มควบคุมอย่างถูกต้อง
