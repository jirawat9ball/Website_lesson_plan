### 1. ⚔️ ส่วนที่ 1: Grid vs Flexbox แตกต่างกันอย่างไร? (15 นาที)
*   **แนวทางการสอนเชิงเปรียบเทียบ**:
    *   **Flexbox (จัดหน้า 1 มิติ)**: เปรียบเหมือน **"รางเดี่ยวเส้นตรงยืดหดได้"** เหมาะสำหรับการเรียงปุ่ม เมนู หรือการ์ดแบบแถวเดียวทิศทางเดียว (แนวตั้งหรือแนวนอนอย่างใดอย่างหนึ่ง)
    *   **Grid (จัดหน้า 2 มิติ)**: เปรียบเหมือน **"กระดานหมากรุกที่มีทั้งแถวนอนและแถวตั้งพร้อมๆ กัน"** เหมาะสำหรับการออกแบบโครงร่างหน้าเว็บใหญ่ระดับพิมพ์เขียว (Page Layouts) หรือนิตยสารที่มีหลายช่องย่อยกระจัดกระจาย

---

### 2. 🏁 ส่วนที่ 2: คอนเซปต์ตารางหมากรุกและแบ่งห้องหับ (display: grid) (15 นาที)
*   **แนวทางการอธิบายเรื่องการสร้างตาราง**:
    *   **Grid Container (ตัวแผงกระดานแม่)**: สั่ง `display: grid;` เพื่อปูพื้นที่ตารางหมากรุกครอบวัตถุด้านใน
    *   **การแบ่งห้อง**:
        *   `grid-template-columns`: สั่งระบุขนาดและจำนวนแนวตั้ง (เช่น `grid-template-columns: 200px 1fr 1fr;` แปลว่า แบ่งเป็น 3 เสาแนวตั้ง เสาแรกกว้าง 200px ส่วนเสาที่สองและสามแบ่งสัดส่วนพื้นที่ที่เหลือเท่าๆ กันคนละครึ่ง)
        *   `grid-template-rows`: สั่งระบุขนาดแถวนอนด้านล่าง
    *   **`gap` (ช่องเว้นวรรค)**: เปรียบเหมือน **"คูเมืองหรือเส้นกั้นแบ่งระหว่างตารางแต่ละช่อง"** เพื่อไม่ให้ข้อมูลเบียดชนกัน

### 🗺️ ส่วนที่ 3: เส้นกริดและการจับจองดินแดน (Grid Lines & Areas) (10 นาที)
*   **แนวทางการสอนแบบเห็นภาพ**:
    *   สอนให้นักเรียนรู้จักการนับเส้นขีดของตาราง (Grid Lines) ซึ่งจะเริ่มนับจากเส้นขอบซ้ายสุดเป็นเส้นที่ 1, 2, 3... เสมอ
    *   **การขยายร่างจองดินแดน (Span)**:
        *   หากนักเรียนมีรูปภาพขนาดใหญ่และต้องการให้รูปภาพกินพื้นที่ครอบคลุม 2 ช่องแนวตั้ง
        *   เขียนสั่งในกล่องลูก `grid-column: span 2;` หรือ `grid-column: 1 / 3;` (กินตั้งแต่เส้นกริดที่ 1 ยาวไปถึงเส้นกริดที่ 3)
    *   การทำความเข้าใจคอนเซปต์นี้จะช่วยปลดล็อกให้นักเรียนสร้างเลย์เอาต์ซ้ายเป็นรูปภาพเด่น ขวาแบ่งเป็นหัวข้อและย่อหน้าข้อความย่อยได้อย่างง่ายดาย

### 4. 🧠 ส่วนที่ 4: กิจกรรมทดสอบความเข้าใจด่วน (Quick Quiz) (5 นาที)
เช็กความพร้อมด้วย 3 คำถามด่วน:
1.  **คำถาม 1**: หากต้องการจัดวางหน้าเว็บบทความนิตยสารที่มีขอบเขตซับซ้อน (มีรูปใหญ่กินพื้นที่ 2 คอลัมน์ และมีข้อความย่อยล้อมรอบ) เครื่องมือใดระหว่าง Flexbox กับ Grid จะเหมาะสมที่สุด? *(แนวตอบ: CSS Grid เพราะสามารถควบคุมทิศทางทั้งแนวนอนและแนวตั้งได้พร้อมกันแบบ 2 มิติ)*
2.  **คำถาม 2**: สัญลักษณ์หน่วย `1fr` ใน CSS Grid ย่อมาจากคำว่าอะไร และทำหน้าที่อะไร? *(แนวตอบ: ย่อมาจาก Fractional Unit ทำหน้าที่แบ่งสัดส่วนพื้นที่ว่างที่เหลือในหน้าจอตามอัตราส่วนยืดหยุ่น)*
3.  **คำถาม 3**: ในตารางกริดแบบ 3 คอลัมน์ เส้นกริดแนวตั้งขวาสุดจะเป็นเส้นหมายเลขใด? *(แนวตอบ: เส้นหมายเลข 4 (เพราะตาราง 3 ช่อง จะมีเส้นขีดกั้นเขตทั้งหมด 4 เส้น))*
