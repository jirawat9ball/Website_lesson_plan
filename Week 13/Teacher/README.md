### 1. 🛋️ ส่วนที่ 1: ยกทัพเฟอร์นิเจอร์สำเร็จรูปมาประกอบร่าง (20 นาที)
*   **แนวทางการเปรียบเทียบเชิงอุปมาอุปไมย (เฟอร์นิเจอร์แต่งบ้านสำเร็จรูป IKEA)**:
    *   การเขียนโครงสร้างแถบเมนูนำทาง หรือสไลด์รูปภาพขึ้นมาเองด้วย HTML/CSS เสมือน **"การเลื่อยไม้ แคะรู ตอกตะปูทำเตียงนอนเองตั้งแต่ต้น"** ซึ่งปวดหัวมากและมีโอกาสพังสูงสำหรับมือใหม่
    *   **Bootstrap Components**: เปรียบเหมือน **"เฟอร์นิเจอร์สำเร็จรูปแกะกล่องชิ้นใหญ่จาก IKEA"** ที่สัดส่วนและหน้าตาระดับหรูหราอยู่แล้ว
        *   **Navbar**: แผงเมนูหัวเว็บติดครบทั้งโลโก้ ช่องค้นหา และปุ่มนำทาง (ย่อขยายจอแล้วกลายเป็นปุ่ม Hamburger สามขีดให้อัตโนมัติ!)
        *   **Cards**: กล่องหุ้มรูปภาพและข้อความคำบรรยาย พร้อมขอบปุ่มกดสวยงาม
        *   **Carousel**: สไลด์โชว์สลับสับเปลี่ยนรูปภาพที่มีปุ่มกดเลื่อนซ้ายขวา

### 2. 🏷️ ส่วนที่ 2: ระบบคลาสวิเศษของ Bootstrap (10 นาที)
*   **แนวทางการสอนเชิงโค้ดสำเร็จรูป**:
    *   Bootstrap คุมความสวยงามผ่านคุณสมบัติชื่อคลาสสำเร็จรูปในระบบเป็นจำนวนมาก เช่น
        *   **คลาสปุ่ม (`btn btn-primary`)**: สั่งตัวเดียวปุ่มกลายเป็นสีฟ้าและมีเอฟเฟกต์สะท้อนแสงทันที!
        *   **คลาสระยะห่าง (`m-3`, `p-2`)**: ย่อมาจาก `margin: 1rem;` และ `padding: 0.5rem;` สะดวกโดยไม่ต้องไปนั่งระบุความห่างทีละพิกเซลในหน้า CSS
        *   สอนนักเรียนเปิดดูตัวอย่างและคัดลอกโครงร่างโค้ดต้นแบบจากเว็บไซต์คู่มือหลักของ Bootstrap สะดวก รวดเร็ว และไม่มีทางเขียนสัญลักษณ์วงเล็บเปิดปิดคร่อมตกหล่น

### 🎨 ส่วนที่ 3: ศิลปะการพ่นสีทับ: Override CSS (10 นาที)
*   **แนวทางการสอนการพ่นสี**:
    *   จะเกิดอะไรขึ้นถ้าบ้านของนักเรียนทุกคนในห้องเรียน ม.5 ดึงข้อมูล Bootstrap มาเหมือนกันหมด? (คำตอบ: หน้าตาเว็บจะเหมือนกันเปี๊ยบ ไม่มีจุดเด่นของแบรนด์ตนเอง)
    *   **Override CSS**: ศิลปะการหยิบเอาเฟอร์นิเจอร์สำเร็จรูปพ่นสีทับใหม่ด้วยสไตล์ของตัวเอง!
    *   **วิธีการลงมือ**: สั่งเขียนคำสั่ง CSS ของตัวเองในไฟล์ภายนอก (เช่น `style.css`) ระบุ Selector ชี้พิกัดไปที่คลาสสำเร็จรูปของ Bootstrap แล้วสั่งเปลี่ยนคุณสมบัติ เช่น เปลี่ยนสีปุ่มเป็นโทนแบรนด์เฉพาะของแบรนด์เรา
    *   *เคล็ดลับเหล็ก*: ลิงก์ไฟล์ `style.css` ของเราไว้ **"ใต้บรรทัดลิงก์ไฟล์ของ Bootstrap CDN เสมอ"** เพื่อให้คำสั่งสุดท้ายของเราทับเส้นทางคำสั่งต้นแบบเดิมได้สำเร็จ

### 4. 🧠 ส่วนที่ 4: กิจกรรมทดสอบความเข้าใจด่วน (Quick Quiz) (5 นาที)
เช็กความพร้อมด้วย 3 คำถามด่วน:
1.  **คำถาม 1**: เหตุใดเราจึงควรลิงก์ไฟล์ `style.css` ส่วนตัวไว้ข้างล่างลิงก์ดึงข้อมูลของ Bootstrap CDN ในโค้ด HTML? *(แนวตอบ: เพื่อให้เบราว์เซอร์อ่านและทำตามสไตล์ของเราทีหลังสุด ซึ่งจะทับ (Override) สไตล์ดั้งเดิมของ Bootstrap สำเร็จรูปได้สำเร็จตามต้องการ)*
2.  **คำถาม 2**: คลาสปุ่ม `btn btn-danger` ของ Bootstrap จะสร้างผลลัพธ์ปุ่มกดออกมาเป็นสีใดตามมาตรฐานสากล? *(แนวตอบ: สีแดง (Danger สื่อถึงการแจ้งเตือนภัย/อันตราย))*
3.  **คำถาม 3**: คลาสระยะห่าง `mt-5` ย่อมาจากคุณสมบัติ CSS คำว่าอะไร และมีทิศทางขยับไปทางใด? *(แนวตอบ: ย่อมาจาก `margin-top` ขนาดระดับ 5 (ผลักระยะเว้นว่างเหนือขอบด้านบนของกล่อง))*
