# สัปดาห์ที่ 2: HTML Forms (แบบฟอร์มและการรับข้อมูล)

## กิจกรรมปฏิบัติ
**[Project] Club SignUp Form (ฟอร์มสมัครเข้าชมรม E-Sports)**

ในสัปดาห์นี้ นักเรียนจะสวมบทบาทเป็นผู้พัฒนาหน้าเว็บใบสมัครออนไลน์ให้กับชมรม E-Sports ของโรงเรียนเพลินพัฒนา โดยจะเขียนโค้ดรับข้อมูลประเภทต่างๆ ของผู้กรอก เพื่อนำไปบริหารจัดการสมาชิกชมรมจริง!

---

### 📋 งานที่ต้องทำ (Core Tasks - 70% คะแนน)
1. **จัดโครงสร้างหัวข้อบทเรียน**: ตั้งชื่อชมรมด้วยแท็ก `<h1>` และคำอธิบายด้วยแท็ก `<p>`
2. **สร้างพื้นที่แบบฟอร์ม**: ครอบช่องกรอกข้อมูลทั้งหมดด้วยแท็ก `<form>`
3. **รับข้อมูลพื้นฐานส่วนตัว**: สร้างช่องกรอก ชื่อ-นามสกุล (`type="text"`), อีเมล (`type="email"`), และรหัสผ่าน (`type="password"`)
4. **ทำฉลากระบุตัวช่องกรอก (Labeling)**: สร้างแท็ก `<label>` คู่กับช่องกรอกแต่ละฟิลด์ และผูกความสัมพันธ์ด้วยการตั้งค่าแอตทริบิวต์ `for` ในฉลากให้ตรงกับแอตทริบิวต์ `id` ในอินพุต
5. **ทำระบบระบุห้องเรียน**: สร้างรายการดึงลงมา (Dropdown Menu) สำหรับเลือกห้องเรียน ม.5/1 ถึง ม.5/4 โดยซ้อนแท็ก `<option>` ไว้ในแท็ก `<select>` ตามโครงสร้าง Nesting ที่ถูกต้อง
6. **สร้างปุ่มส่งแบบฟอร์ม**: ใช้แท็ก `<button type="submit">` เพื่อสั่งจัดส่งข้อมูลสมาชิก

---

### ✨ งานเพิ่มเติมท้าทายความสามารถ (Extra Challenge - 30% คะแนน)
1. **ระบุเครื่องเล่นเกมโปรด**: เพิ่มคำถามเลือกได้เพียงข้อเดียว (Radio Buttons - `type="radio"`) เพื่อระบุแพลตฟอร์มหลัก เช่น PC, Console, Mobile โดยใช้แอตทริบิวต์ `name` ชื่อเดียวกันเพื่อให้ระบบจัดเป็นกลุ่มเดียวกัน (หากเลือกปุ่มอื่น ปุ่มเดิมจะเด้งออก)
2. **เลือกประเภทการแข่งขัน**: เพิ่มคำถามเลือกได้หลายข้อ (Checkboxes - `type="checkbox"`) เพื่อระบุประเภทเกม เช่น MOBA, FPS, Sports
3. **กรอกความสามารถหรือเป้าหมาย**: เพิ่มกล่องกรอกข้อความขนาดยาวด้วยแท็ก `<textarea>` สำหรับนักเรียนกรอกแนะนำตัวเองหรือระบุระดับแรงก์เกมที่ตนเองถนัด
4. **ป้องกันบั๊กและคำใบ้ผู้ใช้ (Attributes)**:
   - ใส่แอตทริบิวต์ `placeholder` เพื่อทำข้อความจางๆ ช่วยบอกตัวอย่างการกรอกข้อมูล
   - ใส่แอตทริบิวต์ `required` บังคับให้กรอกชื่อ อีเมล รหัสผ่าน และระดับชั้น รวมถึงปุ่มวิทยุ (หากไม่กรอกจะไม่สามารถกดส่งใบสมัครได้)
5. **ตกแต่งความสวยงามดึงดูดใจ (Styling)**: เขียนโค้ด CSS เพื่อปรับโทนสีพื้นหลังแบบฟอร์มเป็นธีมเกมเมอร์ยอดฮิต (เช่น ใช้พื้นหลังสีเทาเข้ม ตัวอักษรสีขาว และปุ่มส่งข้อมูลที่มีสีสันสดใส) เพื่อความประทับใจระดับโปร
