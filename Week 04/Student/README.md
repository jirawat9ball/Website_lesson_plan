# สัปดาห์ที่ 4: Introduction to JavaScript

## กิจกรรมปฏิบัติ
[Project] Simple BMI Calculator (โปรแกรมคำนวณ BMI อย่างง่าย)

### งานที่ต้องทำ (Core)
• Core: รับค่าน้ำหนัก (กิโลกรัม) และส่วนสูง (เซนติเมตร) แล้วแสดงผลค่า BMI เมื่อคลิกปุ่มคำนวณ
- ใช้ `document.getElementById` ดึงค่าน้ำหนักและส่วนสูง
- ใช้ `parseFloat` ในการแปลงประเภทข้อมูลเป็นตัวเลขทศนิยม
- คำนวณ BMI = น้ำหนัก / (ส่วนสูงหน่วยเมตร * ส่วนสูงหน่วยเมตร)
- แสดงผลลัพธ์ผ่าน `innerText` ของหน้าเว็บ

### งานเพิ่มเติม (Extra/Review)
• Extra: นำผลลัพธ์ BMI ที่ได้ ไปเปรียบเทียบเกณฑ์สุขภาพเบื้องต้นและแสดงข้อความบอกระดับ (เช่น น้อยกว่า 18.5 = ผอม, 18.5-22.9 = ปกติ, 23 ขึ้นไป = อ้วน)
